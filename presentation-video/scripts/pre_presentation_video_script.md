# DoctaRx Pre-Presentation Video Script

## Purpose

This video opens the Federal Ministry presentation before Jonah Baka begins the formal PowerPoint. It is not a founder avatar, not a sales advert, and not a government approval scene.

The purpose is to make the Abuja pilot program feel obvious, practical, and achievable.

## Core Message

Start small. Connect selected PHCs, one General Hospital, doctors, pharmacy partners, and a Ministry dashboard. Measure results. Improve workflows. Scale only after evidence.

## Structure

1. Show the current patient journey challenge in Abuja.
2. Introduce a small Abuja pilot, not a national rollout.
3. Show the participating actors: patient, PHC nurse, doctor, pharmacy partner, referral officer, Ministry dashboard user, and DoctaRx support.
4. Show the workflow from request capture to follow-up visibility.
5. Show the live portal walkthrough using demo data only.
6. Hand over to Jonah Baka for the formal pilot roadmap and partnership discussion.

## Final Line

This is the care journey DoctaRx proposes to strengthen. Jonah Baka will now present the pilot roadmap and partnership details.
